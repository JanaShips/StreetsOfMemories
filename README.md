# Streets of Memories - Quote Images

Image assets for the Streets of Memories quotes app. Served via jsDelivr CDN.

## How to add images

1. Navigate to a category folder (e.g. `motivation/`)
2. Click **Add file** > **Upload files**
3. Drag your images and commit
4. The GitHub Action auto-generates `manifest.json`

## Folder structure

| Folder | Category |
|---|---|
| `good-morning/` | Good Morning |
| `good-night/` | Good Night |
| `inspire/` | Inspire |
| `fun/` | Fun |
| `love/` | Love |
| `motivation/` | Motivation |
| `special/` | Special |
| `thought-of-the-day/` | Thought Of The Day |
| `birthday/` | Birthday |
| `anniversary/` | Anniversary |

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
